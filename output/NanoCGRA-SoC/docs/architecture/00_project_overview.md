# NanoCGRA-SoC

Version: v0.1

## Overview

NanoCGRA-SoC is a minimal RISC-V based System-on-Chip designed for
first-silicon success on GF180MCU.

The project focuses on:

- Extremely small die area
- Simple architecture
- Easy verification
- Memory-mapped peripherals
- CGRA acceleration

Current implemented components:

- SMPB Interconnect
- UART Peripheral
- Address Decoder
- Default Slave
- Read/Ready/Error Multiplexers

Future components:

- FAZYRV RV32I CPU
- SRAM
- CGRA
- GPIO
- Timer

Target Technology:

GF180MCU

Target Die Size:

0.25 mm × 0.25 mm

Target Frequency:

20 MHz
